# 项目概述

## 简介

Nexus OS 是一个跨平台的个人数据管理系统，旨在为用户提供统一的文件访问与数据同步能力。通过聚合多个云存储服务，用户可以在任意设备上无缝访问自己的文件和数据。

## 设计理念

**"文件与数据分离"** - 大文件存储在云端网盘，结构化数据存储在轻量数据库，实现高效的资源管理与多端同步。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **文件网关** | Alist | 聚合阿里/百度/夸克等网盘，提供 WebDAV/HTTP 接口 |
| **数据后端** | PocketBase | 轻量级后端，内置 SQLite 数据库与 REST API |
| **容器化** | Docker Compose | 一键部署双后端服务 |
| **客户端** | Swift / Kotlin | macOS、iOS、Android 原生应用 |

## 核心功能

1. **多网盘聚合** - 通过 Alist 统一管理阿里云盘、百度网盘、夸克网盘等
2. **阅读进度同步** - 跨设备同步阅读位置与状态
3. **笔记与标签** - 结构化管理个人笔记和文件标签
4. **离线访问** - 支持文件缓存与离线阅读
5. **多端同步** - macOS、iOS、Android 数据实时同步
