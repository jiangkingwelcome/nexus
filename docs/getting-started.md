# 快速开始

## 环境要求

- Node.js >= 18.0.0 (或其他运行时)
- [其他依赖]

## 安装步骤

### 1. 克隆项目

```bash
git clone [仓库地址]
cd Nexus
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量

复制环境变量模板：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填入相应配置。

### 4. 启动项目

**开发模式：**

```bash
npm run dev
```

**生产模式：**

```bash
npm run build
npm start
```
